# Diretrizes de Contribuição

[cite_start]Todas as contribuições, relatórios de bugs, correções de bugs e melhorias são bem-vindas. [cite: 914]

Antes de submeter, por favor:
1. Abra uma Issue para grandes mudanças.
2. Garanta que seu código siga as diretrizes de estilo do projeto.
3. Mantenha os Pull Requests focados em um único problema.
